# Sudoku

#### Para compilar
```console
make
```

#### Para ejecutar todos los tests
```console
make test
```

#### Para ejecutar todos los tests con valgrind
```console
make valgrind
```

#### Para ejecutar y generar 3 tableros Sudoku aleatorios con solución única
```console
make run
```

#### Para ejecutar indicando el archivo deseado
```console
./testeador <archivo_entrada>
```
